Introduction

Matplotlib : c'est une librairie qui permet de tracer des graphes (dans le sens graphiques) : inspir de Matlab au dpart. inclus 2 faons de l'utiliser : soit avec une approche objet : plus complique et plus verbeuse, mais plus volue. soit via des appels de fonctions, avec pyplot : plus simple. pyplot fournit ainsi des raccourcis qui vitent la formulation objet plus longue crire. permet de faire des graphes qui peuvent tre comltement adapts si besoin. Sur une figure, on peut tracer plusieurs graphes. permet aussi de dessiner. matplotlib rend ainsi possible la cration de graphes l'intrieur d'applications complexes autorises par le langage python, et ceci sans quitter le langage python. Notions principales pour les graphes sous matplotlib : figure : une figure qui peut tre sauve dans un fichier. axe : un graphe (graphique) individuel appartenant une figure qui peut en compter plusieurs. Attention, ce n'est pas un axe de coordonnes ! axis : un axe de coordonnes appartenant un objet axe. divers objets graphiques qui drivent de la classe Artist et qui participent au graphe. Ce sont par exemple des rectangles, des lignes, du texte. Ces diffrents objets peuvent tre totalement configurs (y compris individuellement) pour adapter le graphe aux besoins en appelant dessus des mthodes pour modifier leurs proprits !

Matplotlib est probablement le package Python le plus utilis pour les graphiques 2D. Il fournit la fois un moyen rapide de visualiser les donnes de Python et des chiffres de qualit de publication dans de nombreux formats. Nous allons explorer matplotlib en mode interactif couvrant les cas les plus courants.

pyplot et gnralits Importation du module : from matplotlib import pyplot. Pour tracer un graphe x-y avec les points relis (pour un nuage de points, utiliser plutt scatter) : from matplotlib import pyplot : pour importer le module. pour tracer un graphe avec des valeurs de x et des valeurs de y en reliant les points dans l'ordre de la liste : pyplot.plot([1, 2, 3, 6], [1, 4, 9, 36]) pyplot.show() : montre le graphe courant. on peut indiquer le symbole et la couleur : pyplot.plot(x, y, 'bo') : bleu et avec des ronds. on peut aussi indiquer tous les paramtres avec des tags spars : pyplot.plot(x, y, color = 'green', linestyle = 'dashed', linewidth = 2, marker = 'o', markerfacecolor = 'blue', markersize = 5) : symboles ronds en bleu avec une taille de 5 et traits pointills en vert. pyplot.plot(x, y, label = 'A') : permet de mettre un label sur la courbe !!! on peut tracer plusieurs courbes sur le mme graphe : pyplot.plot(x, y, 'r+', y, y, 'bo') si on a un objet AxesSubplot, on peut tracer dessus un graphe en faisant : axes.plot(...)

import numpy as np
import matplotlib.pyplot as plt

# Create a figure of size 8x6 inches, 80 dots per inch
plt.figure(figsize=(8, 6), dpi=80)

# Create a new subplot from a grid of 1x1
plt.subplot(1, 1, 1)

X = np.linspace(-np.pi, np.pi, 256)
C, S = np.cos(X), np.sin(X)

# Plot cosine with a blue continuous line of width 1 (pixels)
plt.plot(X, C, color="blue", linewidth=1.0, linestyle="-")

# Plot sine with a green continuous line of width 1 (pixels)
plt.plot(X, S, color="green", linewidth=1.0, linestyle="-")

# Set x limits
plt.xlim(-4.0, 4.0)

# Set x ticks
plt.xticks(np.linspace(-4, 4, 9))

# Set y limits
plt.ylim(-1.0, 1.0)

# Set y ticks
plt.yticks(np.linspace(-1, 1, 5))

# Save figure using 72 dots per inch
# plt.savefig("exercise_2.png", dpi=72)

# Show result on screen
plt.show()

import numpy as np
import matplotlib.pyplot as plt

n = 12
X = np.arange(n)
Y1 = (1 - X / float(n)) * np.random.uniform(0.5, 1.0, n)
Y2 = (1 - X / float(n)) * np.random.uniform(0.5, 1.0, n)

plt.axes([0.025, 0.025, 0.95, 0.95])
plt.bar(X, +Y1, facecolor='#9999ff', edgecolor='white')
plt.bar(X, -Y2, facecolor='#ff9999', edgecolor='white')

for x, y in zip(X, Y1):
    plt.text(x + 0.4, y + 0.05, '%.2f' % y, ha='center', va= 'bottom')

for x, y in zip(X, Y2):
    plt.text(x + 0.4, -y - 0.05, '%.2f' % y, ha='center', va= 'top')

plt.xlim(-.5, n)
plt.xticks([])
plt.ylim(-1.25, 1.25)
plt.yticks([])

plt.show()
